# 测试脚本
- 当前目录下：start_test.sh 、stop_test.sh
# 测试说明
- 本次测试是测试RK3568 ubuntu 系统板子上4个网口和一个光纤口的网络性能。
- 光口eth0：192.168.8.200
- 网口eth2：192.168.2.200
- 网口eth3：192.168.3.200
- 网口eth4：192.168.4.200
- 网口eth5：192.168.5.200
# 测试方法
1. 在RK3568端使用 ./start_test.sh 开启脚本。
2. 分别使用5台PC电脑，设置好对应的静态网段后，使用iperf3工具，执行命令
iperf3 -c 192.168.X.200 -p 5202 -u -b 200M -P 4 -t 600，其中X是对应的网段。
3. 等测试结束后，使用./stop_test.sh 停止RK3568的测试脚本。
4. 在终端test_logs_*下会生成本次测试的日志等文件。现在已经放在了./test_logs_20260416_183338 中了。
